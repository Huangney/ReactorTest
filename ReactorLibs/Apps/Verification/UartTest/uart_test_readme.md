# UartTest 使用说明

`UartTest` 是一个用于 UART 性能与压力测试的应用程序，主要用于验证 `BspUart` 的 FIFO 发送机制以及 `UartPortor` 的多节点数据分发能力。

## 适用场景
当你修改了底层的 UART 驱动（例如加入了 FIFO 异步发送支持），或修改了中间的协议包装层时，可以通过此应用快速进行压力测试，验证数据发送是否会丢包、底层缓存是否充足、系统状态是否稳定。

## 如何使用

`UartTest` 继承自 `Application` 并已实现为单例。你**不需要**手动调用它的 `Start()` 或 `Update()` 方法，只需要将其注册到 `System` 框架中即可全自动运行。

在你的 `MainFrame` 或系统初始化入口处，添加以下注册代码：

```cpp
#include "UartTest.hpp"
#include "System.hpp"

// 在系统初始化阶段注册该应用
System::GetInstance().Register(&UartTest::GetInstance());
```

## 测试流程解析
注册成功并启动系统后，测试会自动经历以下四个阶段，在此期间你可以通过 `BspLog` 串口日志查看当前进度：

1. **准备阶段 (PREPARE)**：闲置等待 1 秒钟，确保系统其他模块初始化完毕。
2. **带宽持续测试 (BANDWIDTH_TEST)**：对发送通道进行为期 10 秒钟的满功率压测验证。
3. **FIFO 突发接纳测试 (FIFO_TEST)**：为期 2 秒的 FIFO 瞬时接纳能力压测。每隔 50ms 会瞬间向队列中灌入 15 个连续数据包（约 1050 字节），以测试底层 FIFO 队列的平滑缓冲能力。
4. **完成阶段 (DONE)**：所有测试完毕，自身停止发送并进入静默闲置状态，仅每隔 5 秒打印一次存活日志。

> **提示**：测试数据默认使用 PortID `0x10`，通过 `Hardware::huart_host` 发送长度为 64 字节的有规律假数据包（首字节为 'A'）。你可以通过外部抓包工具配合检查数据包是否完整与连续。
